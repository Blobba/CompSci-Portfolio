Hardware

Technology is made up of many different parts that all have their own jobs within the product.

This is a motherboard.

A motherboard is the part of the computer that allows all the parts to recieve power and communicate with one another

This is a cpu

A cpu is a component in a computer that processes instructions that it gathers from decoding the code in programs and other similar files.

This is RAM

RAM stands for random access memory.

RAM is considered "random access" because you can access any memory cell directly only if you know the row and column that intersects at that cell.

This is a hard drive

a hard drive is something that reads, writes, and saves data on a computer.

This is a power supply

It is considered a tranformer because it transforms electricity from the wall into usable power for the computer.
